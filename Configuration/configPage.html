Help
  • Primary — The main poster. Vertical cover art, shown everywhere: library grid, search results, item page.
  • Logo — The title as a transparent PNG graphic (stylized text/wordmark). Displayed overlaid on backgrounds in detail views.
  • Backdrop — Full-width background image shown behind the item detail page. Multiple backdrops can exist and rotate.
  • Banner — Wide horizontal strip image (typically 758×140). Used in some clients as an alternate list/header view.
  • Thumb — Wide landscape thumbnail (16:9). Used in episode lists and some card layouts instead of the vertical poster.
  • Clearart — Character or scene artwork on a transparent background. Decorative, shown in some skins alongside metadata.
  • Disc — A circular image of the physical disc (DVD/Blu-ray label art). Mostly cosmetic, shown in detailed metadata views.
  • BoxRear — The back of the physical box/case. Informational, rarely displayed by clients — mostly for completionists.
